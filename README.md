# BizCity Group - Workspace Catalogue

A digital catalogue of BizCity Group's serviced offices, coworking and meeting spaces across three landmark Dubai addresses.

## Locations
- **Biz City** - City Tower 2, 14th Floor, Sheikh Zayed Road, Dubai
- **Biz Elite** - Office 2701, 27th Floor, Sheikh Rashid Tower, DWTC, Dubai
- **Biz Platinum** - 21st Floor, Sheikh Rashid Tower, DWTC, Dubai

## View it
- `index.html` - the main catalogue (open in any browser).
- `biz-city.html`, `biz-elite.html`, `biz-platinum.html` - per-location lookbooks.
- Save a PDF anytime via the **Save PDF** button or Ctrl+P.
